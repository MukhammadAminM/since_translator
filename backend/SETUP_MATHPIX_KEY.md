# Быстрая настройка Mathpix API ключа

## Шаг 1: Откройте файл `.env` в папке `backend/`

Если файла `.env` нет, создайте его.

## Шаг 2: Добавьте ключи Mathpix

Добавьте следующие строки в файл `.env`:

```env
MATHPIX_APP_ID=your_app_id_here
MATHPIX_APP_KEY=15c7f5976cedd9293c0d974902efe991345bfd462edb8ea2d6c28e7a44c1c90e
```

**Важно:** Обычно Mathpix требует два ключа:
- `MATHPIX_APP_ID` - ID приложения
- `MATHPIX_APP_KEY` - ключ приложения (уже предоставлен)

Если у вас есть только один ключ, попробуйте:
1. Использовать его как `MATHPIX_APP_KEY`
2. Оставить `MATHPIX_APP_ID` пустым или использовать тот же ключ
3. Или получить `APP_ID` на https://accounts.mathpix.com/

## Шаг 3: Перезапустите сервер

После добавления ключей перезапустите сервер, чтобы система загрузила новые переменные окружения.

## Проверка

После перезапуска сервера вы должны увидеть в логах:
```
✅ Mathpix API настроен
```

Если видите:
```
⚠️  Mathpix API не настроен. Установите MATHPIX_APP_ID и MATHPIX_APP_KEY в .env
```

То проверьте:
1. Правильность имен переменных (MATHPIX_APP_ID и MATHPIX_APP_KEY)
2. Что файл `.env` находится в папке `backend/`
3. Что после `=` нет пробелов
4. Перезапустили ли вы сервер
